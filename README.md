# nesti 🌿
### The AI-powered newborn tracking app for modern Indian parents

Built for parents of preterm and newborn babies who need more than 
just a logging tool — they need a care companion.

**Live app:** https://nesti-app.vercel.app

## Features
- Daily tracking: feeds, sleep, diapers, medicines
- AI Care Assistant powered by Claude
- Growth charts with WHO percentile tracking (corrected age)
- Doctor visit summary generator
- Voice logging
- Soothing sounds library
- Vaccination tracker

## Built with
React · Vercel · Claude API · Recharts · Web Speech API

## The story
Maanvik was born at 30 weeks + 4 days. 
This app was built for him — and for every parent who needed 
something like this and couldn't find it.
